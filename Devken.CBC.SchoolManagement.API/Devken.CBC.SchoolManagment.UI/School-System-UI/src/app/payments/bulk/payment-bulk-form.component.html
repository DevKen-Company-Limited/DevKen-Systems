help_outline How to use Payment Entry — works for 1 student up to your entire school
person
Single Student
  1. 1 Fill in Shared Settings (date, method).
  2. 2 Click Add Row → search and select the student.
  3. 3 Pick the invoice, enter amount → Submit.
group
Multiple Students
  1. 1 Fill in Shared Settings and set an Invoice Strategy.
  2. 2 Open Import by Class, filter to the class → tick students → Import.
  3. 3 Invoices auto-selected. Adjust amounts per row → Submit.
corporate_fare
Whole Class / School
  1. 1 Set Invoice Strategy to "First unpaid" or "Highest balance".
  2. 2 Open Import by Class. Leave filters blank. Set a Default Amount, tick "Select all" → Import.
  3. 3 Review rows, fix any flagged in amber → Submit all.

tips_and_updates Useful features to know

auto_fix_high Invoice Strategy — Set "First unpaid", "Highest" or "Lowest" in Shared Settings. Invoices auto-picked on import. Hit Re-apply to all rows to update existing rows instantly.
content_copy Copy invoice to all rows — Once a row has an invoice selected, click the copy icon in its header to push that same invoice to every other row.
search Student search — In each row's Student dropdown, start typing a name or admission number. The list filters live with no scrolling needed.
attach_money Default amount on import — Set a "Default Amount (KES)" in the Import panel before importing. All rows will be pre-filled — still editable after.
refresh Retry failed payments — After submission, if some rows fail, click Retry Failed to reload only those rows without re-entering successful ones.
radio_button_checked Row status dots — not started, in progress, complete, error.
